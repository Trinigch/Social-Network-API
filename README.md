# Social-Network-API

## Description
API for a social network web application where users can share their thoughts, react to friends' thoughts, and create a friend list. 


## Technologies

-Typescript
-JavaScript
-node.js
-Express.js
-MongoDB

## Installation

npm install 

## Usage

To start the application, run the following command:

-npm run start
